#Sistema de Gestión Hotelera

Programa hecho en Python con MySQL para administrar huéspedes, habitaciones y reservas.

Permite registrar huéspedes, ver habitaciones disponibles, crear y cancelar reservas.
El sistema valida fechas y calcula automáticamente el costo total según las noches.

Estructura básica:

main.py (programa principal)

testeo_conexion.py.py (prueba de conexion)

conexion/base_data.py (conexión a base de datos)

clases/ (clases de huésped, habitación y reserva)

base_datos/base_data.sql (script de base de datos)

Requisitos:

Python 3.7 o superior

PyMySQL

XAMPP o WampServer

Para ejecutar:

python main.py
